熟悉命令行

命令行作为计算机科学中程序的最开端，我个人认为是应当掌握的部分，并且会对你们的未来有非常大的帮助。所以本作业的目的是为了让大家熟悉命令行的使用，并亲身去解决一些奇奇怪怪的问题，填平这些坑，你也能成为一个大神。

首先我们要明白 命令行 是一个什么东西。命令行其实也就是黑框框，而这个黑框框允许我们对系统进行一些操作。以Windows系统为例，同时按下"Win+R"键，输入"cmd"，就可以看到一个黑框框。这个黑框框内，最左边，是盘符（C、D、E、F之类的），然后是自己的根目录所在的位置，在>符号之后，会有光标在闪动。

我们可以输入"abcde"，可以看到，系统提示"abcde"并不是系统内部或外部命令，也不是程序，所以运行失败。可是当我们输入"dir"的时候，我们神奇的发现，所有这个目录下的文件夹被显示了出来，说明"dir"命令执行成功了，之后我们输入"cd (某一个你的文件夹的名字)"(注意中间有空格)后，我们发现我们进入到了这个文件夹中，再输入"cd .."，我们又回到了原来的文件夹。有没有感受到命令行的魔力！！！（好吧我知道你们没有╮(╯_╰)╭）

    GCC编译器安装

    当各位同学安装完Linux后，就可以打开进行瞎操作了。一段操作之后，然后我们就需要各位做点正事了，安装C/C++的编译器。理论上大多数Linux系统安装之后都已经预装了GCC编译器，各位同学可以打开Linux系统的命令行（也就是终端——Terminal），输入以下命令查看。 

    gcc --version
    如果出现了对应gcc的版本号，那就说明gcc已经预装在系统中了。如果出现command not find的话，请跟随下列方法安装gcc。

  • Ubuntu

    • apt-get update
apt-get install gcc

  • CentOS

    • yum update
yum install gcc

  • Mac OS X

    • brew update
brew install gcc

  • 其他系统

    • 对于使用其他系统的同学，可以查找本系统对应的应用安装命令，用上述同样的方式来安装gcc编译器。

    第一个Linux程序

    打开命令行，我们首先建立一个文件夹。

    mkdir FisrtProgram

    然后我们进入到这个文件夹中。

    cd FisrtProgram

    然后我们使用Linux系统自带的gedit编辑器，建立一个C++文件，输入以下命令。

    gedit HelloLinux.cpp

    此时可以看到打开了一个编辑器，我们在其中输入以下代码，按ctrl+s保存文件。

    #include <iostream>

using namespace std;
                
int main()
{
    cout << "Hello Linux!" << endl;
    return 0;
}

    代码编辑完成后，我们可以编译这个文件了。

    g++ HelloLinux.cpp

    然后我们发现，在这个文件夹中多了一个名为a.out的文件，这个就是我们编译成功的可以运行的文件了，输入下列命令可以看到标准输出中打印出了"Hello Linux!"。

    ./a.out

    当然了，在这个过程中可能会遇到很多问题，请各位同学善用搜索器来填平这些坑。
