Project 2 类的使用

在学习一个东西的时候，上来就想如何实现它，那必然是非常难的。首先我们应该了解怎么使用它，对它的使用有了较深的理解之后，我们才能够从用户的角度，来更好地实现它。那么本章，我们来实验下如何使用一个别人已经写好的类。

Source:
  • 本章用到的头文件：Sales_item.h

这个头文件非常简单，包含了一个Sales_item类，我已将这个头文件所能实现的操作罗列如下，也希望各位能够看懂这个头文件中所写的东西是什么意思。

操作符 作用
>> 输入
<< 输出
+ 相加
+= 相加并赋值
== 比较是否相等
!= 比较是否不相等
isbn() 返回ISBN号
avg_price() 返回均价
