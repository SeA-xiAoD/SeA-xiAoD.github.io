Project 1 回顾C语言基础语法

我知道大家都学过C语言，那C++曾经被人称作“带类的C”，当然，这个说法肯定是不准确的，但是体现了C++对于C语言的延续性。所以我们首先需要回顾一些和C语言相关、相似甚至相同的基础性语法。

Assignment:

  • 1. 编写程序，使程序能够在输入两个整数（int范围内），在标准输出中打印出两个整数之和。

  • 2. 编写程序，使用while循环将50到100的整数相加，并输出和。

  • 3. 编写程序，输入两个整数，第一个整数小于第二个整数，使用for循环，打印出两个数之间的所有整数。

  • 4. 编写程序，任意输入两个数，使用for循环，打印出两个数之间的所有整数。

  • 5. 编写程序，使用if条件句，输入一个整数，按照如下表格打印出对应等级。

    • 分数范围 对应等级
    0 - 59 不及格
    60 - 79 中等
    80 - 89 良好
    90 - 100 优秀
    其他 输入不合法
Consideration:

  • 1. for循环和while循环各有什么优缺点？

  • 2. 指针与引用的区别是什么？

    • #include <iostream>

using namespace std;

int i = 1;

int main()
{
    cout << i << endl;
    int i = 2;
    cout << i << endl;
    for(int i = 3; i <= 10; i++)
    {
        cout << i << endl;
    }
    cout << i << endl;
    return 0;
}
