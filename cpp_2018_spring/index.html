课程设计

课程设计要求：
  • 1. 分组要求。课程设计将会对同学们进行分组，每个组最多3-4人，为了提升大家的交际与合作能力，每组必须至少有一位女生，可自由组合，并由组长向助教汇报分组情况，鼓励不熟悉的同学间相互组合、相互帮助。
  • 2. 诚信要求。不得抄袭，发现抄袭者以零分计算。有自己创意性的idea可给予加分。可以修改开源代码，但是需要有自己的见解与想法，且此条不鼓励。
  • 3. 技术要求。要求有GUI( Graphic User Interface)界面，通俗说就是不能是单纯的黑框框程序，大家可以自由调用开源API( Application Programming Interface)，如 Windows 10 SDK、 MFC、 QT、 OpenGL、 OpenCV等。
评分规则：
  • 1. 课程设计评分由期末汇报与课程设计质量共同评估。
  • 2. 汇报期间，小组成员将接受老师及同学们的提问，且每位同学都需要回答问题，本小组的分数将由提问期间表现最差的同学决定。设置这样的评估方式或许会对优秀的同学有不公平的情况， 但是对各位的要求是显而易见的，我们希望各位同学互相帮助，学会团队协作，不要让任何同学掉队，这才是设置小组这样一个集体的意义。
